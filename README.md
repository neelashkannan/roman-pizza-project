# Roman Pizza Project

Authentic Roman Pizza Al Taglio - Edinburgh

## About

A minimal, premium website for Roman Pizza Project, featuring:

- **72-hour fermented dough** crafted with heritage grains
- **Pizza al taglio** - traditional Roman-style pizza cut to order
- **Iran flag color scheme** - elegant green, white, and red accents

## Tech Stack

- Next.js 15 - React framework with App Router
- TypeScript - Type-safe development
- Tailwind CSS - Utility-first styling
- Google Fonts - Inter and Playfair Display
